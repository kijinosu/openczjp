# openczjp

[![Copier](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/copier-org/copier/master/img/badge/badge-grayscale-inverted-border-teal.json)](https://github.com/copier-org/copier)

In 1987, the USDA's Economic Research Service (ERS) and its partners at
land grant universities developed 'Commuting Zones' (CZ's) and 'Labor
Market Areas' (LMA's) \[tolbert1987labor\] as delineated geographic
areas for measuring integrated labor processes.

## Requirement

## Install this package

    pip install openczjp
